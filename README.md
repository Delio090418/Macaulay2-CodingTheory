# Introduction

The `CodingTheory` package for Macaulay2 was created by various authors during a Macaulay2 workshop organized by Cleveland State University held during the summer of 2020.

## Tutorial: How to make a pull request to this repository

If you want to make a change to this repository, the procedure is as follows:

1. Create a fork of the new repo by clicking the "Fork" button on this page.
2. Go to your personal GitHub profile and clone the fork that you just made to your computer.
3. Make your changes to the code.
4. Commit your changes locally.
5. Push your commit to your forked repository on GitHub.
6. In your browser, submit a pull request by clicking the "Request pull" button on your fork's page.

[Here](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request-from-a-fork) is a link to some documentation of this process containing specific commands.
